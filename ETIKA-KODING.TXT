1. Database
	- menggunakan migrate
	- setiap tabel untuk id primary pakai "id auto_increment"
	- terapkan realtionship
	- terapkan created_at dll jika tabel membutuhkan time
	- menamai
		- Master "m_kelas, m_siswa"
		- Riwayat "r_kelas, r_absen"
		- Role "m_role",

2. Model
	- menamai model harus sama dengan tabel dalam database
	- jika ada tabel/model yang berelasi terapkan eloquent relationship
	- table yang tidak membutuhkan created_at dll. gunakan ini "public $timestamps = false"
	- jika ada table yang mempunyai kolom gambar/image/foto dll. buatkan fungsi model yg berisikan html image dengan nama fungsi sesuai kolom
		- foto -> funtion imageFoto($className){
			return "<img src='".Config::app("image_upload").$this->foto."' class='{$className}'/>";
		}
3. Controller
	- penulan controlller gunakan Kapital C
	- untuk laravel 9, gunakan konsep CRUD dari laravel (Resource)
	
		php artisan make:controller CSiswa --resource
		Route::resource("siswa",CSiswa::class); 
		Route::get("detail-lokasi/{lokasi}",[Lokasi::clas,'detail'])

	- untuk logika implementasikan konsep repository laravel
	- untuk fungsi yg mengandung Datatable jadikan 1 Controller saja yakni CDatatable
3. View
	- untuk memetakan file jadi perfolder
	- contoh :
		- Master
		  |- Siswa
		     |- index (Gunakan Datatable)
		     |- form (gunakan file ini untuk update/create)
		     |- detail (jika di butuhkan)
		- walaupun frontend nya hanya menggunkan 1 file, masukkan di folder saja
	- Pengelompokan File Template Frontend
		- app.blade.php
		- panels
		  |- header.blade.php
		  |- css.blade.php
		  |- navbar.blade.php
		  |- sidebar.blade.php
		  |- content.blade.php
		  |- footer.blade.php
		  |- js.blade.php
		- untuk menamai file harus menggunakan huruf kecil semua dan spasi gunakan underscor "_"
		- gunakan if else untuk mengekekusi file manakah yg di butuhkan
4. Variable
	- Global
		- gunakan awalan underscor dan di sambung dengan huruf Capital
	- Private
		- gunakan huruf kecil di awalan dan huruf besar di kalimat ke dua contok $masterSiswa
5. Fungsi 
	- namai fungsi dengan huruf kecil dan underscor untuk spasi
	- contoh : funtion master_siswa(){}
